🔑 Spawn Vendo Key Fobs — blank, bind later

Codes SKV-0001 … SKV-0050, three per vendo. Write the vendo name in the big NAME box. Print at 100% / Actual size. Cut on the dashed line.
How to use: stick the three fobs on a vendo's Duplicate, Pungpung, and Board keys. Write the vendo name in the NAME box — it now has two lines and full width for long names. Later, in Spawn Keys → Bind, scan the vendo then scan its three fobs to link each code to the vendo and its key type.

Same 18mm Data Matrix & scanner as the area (SK01–SK20) fobs. Scan with flash OFF at ~10cm; tilt slightly if the plastic shines.

Data Matrix — 18mm · big NAME box · three per vendo

50 vendos × 3 = 150 fobs. Each code unique and registered, ready to bind.
