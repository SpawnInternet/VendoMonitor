🔑 Spawn Vendo Key Fobs — blank, bind later

5-letter codes, three per vendo. Free space below to write the vendo name. Print at 100% / Actual size. Cut on the dashed line.
How to use: stick the three fobs on a vendo's Duplicate, Pungpung, and Board keys (colored chip tells them apart), and write the vendo name in the free space. Later, in Spawn Keys → Bind, scan the vendo then scan its three fobs to link each code to the vendo and its key type.

Same 18mm Data Matrix & scanner as the area fobs. Scan with flash OFF at ~10cm; tilt slightly if the plastic shines.

Data Matrix — 18mm · 5-letter codes · three per vendo

50 vendos × 3 = 150 fobs. Each code unique and registered, ready to bind.
