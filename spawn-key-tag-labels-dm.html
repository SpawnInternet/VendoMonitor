🔑 Spawn Key Tags — Data Matrix

Print at 100% / "Actual size". Never "Fit to page" — that rescales the code.
Why these replaced the Code128 tags. The old barcode packed 79 modules across 24mm, giving a 0.30mm narrowest bar. Phones without autofocus need about 0.50mm. That is why it hunts and blurs — the tag was below what the camera can resolve.

Data Matrix needs only 12 modules instead of 79, so the same tag gets a 0.67–1.1mm module: 2–3× larger. It also carries error correction, so a scuffed or half-dirty tag still reads. Code128 has none — one damaged bar and it is dead.

Print black on white, matte. Glossy laminate throws flash back at the lens and is the second most common cause of failed scans.
