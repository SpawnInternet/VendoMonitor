⬆ Upgrade Supabase
How to free up space — archive 2025 data:
1. Click 📦 Backup Now first — always backup before deleting
2. Go to Supabase SQL Editor and run this to remove old skipped rows:
DELETE FROM transactions WHERE date < '2026-01-01' AND is_skipped = true; This removes ~30-40MB of duplicate/skipped rows from 2025.
3. If you need more space, export 2025 active data to CSV first, then delete:
DELETE FROM transactions WHERE date < '2026-01-01';
